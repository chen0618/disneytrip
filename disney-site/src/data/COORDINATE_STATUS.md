## Notes for Future Sessions

1. **How to verify a coordinate**: Use OpenStreetMap Overpass API:
   ```
   curl -s "https://overpass-api.de/api/interpreter" --data-urlencode "data=[out:json];(node[\"name\"=\"VENUE NAME\"](28.3,-81.7,28.5,-81.4);way[\"name\"=\"VENUE NAME\"](28.3,-81.7,28.5,-81.4););out center;"
   ```

2. **When adding new locations**: Always use OSM Overpass API to find real coordinates. Mark as "OSM" in this file with the date.

3. **If OSM has no result**: Try alternate names, or use the Nominatim geocoder as fallback. If using a manual estimate, mark as "estimate" here.

4. **Coordinate precision**: Use at least 5 decimal places (e.g., 28.41772, not 28.418).
